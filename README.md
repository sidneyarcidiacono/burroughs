# burroughs
### A text-generating neural network that simulates question and response with William Burroughs
#### Powered by Brain.js

#### To Run:

Create your virtual environment:

```zsh
python3 -m venv env
```

Install dependencies:

```zsh
pip3 install -r requirements.txt
```

Activate:

```zsh
source bin/env/activate
```

Run:

```zsh
python3 app.py
```
